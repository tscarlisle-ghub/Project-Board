Your data saves to tscarlisle-ghub/project-board on GitHub.
Paste your Personal Access Token below. It stays in this browser only — never sent anywhere except GitHub's API.
How to get a token: github.com → profile photo → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token → check repo scope → copy the token.

First time? After connecting, click the sync button or make any change — your data will be saved to data.json in your repo and will load on any device that opens this page.
